How Agent Referrals Work

Complete Referral Flow

Understanding the end-to-end process of how agents earn commissions through the Online IBS referral system.

Agent Receives Unique Code
Upon approval, agents receive region-specific referral codes (e.g., AGENT-JOHN-WESTERNEUROPE). Each agent can have multiple codes for different regions they serve.
Agent Shares Code
Agent promotes their code through marketing materials, social media, direct outreach, educational fairs, or their professional network. The code can be shared via link or entered manually at checkout.
Student Uses Code at Signup or Checkout
When a student registers with the referral code or applies it during payment, they are permanently linked to that agent. Regional discounts are automatically applied based on the code's region.
Student Completes Purchase
Student selects a program, proceeds to checkout, and completes payment via Stripe. The system records the transaction with the agent's attribution.
Commission Calculated & Credited
System automatically calculates commission based on the agent's current tier (15-30%) and the course price. Commission is instantly credited to the agent's balance.
Agent Requests Payout
Agent can request payout anytime from their dashboard. Payouts are processed within 24 hours via Stripe Connect (instant) or bank transfer (1-3 business days).

How Student Referrals Work

Step-by-Step Process

Every enrolled student receives a unique referral code that they can share with friends and family to earn credits.

Student Gets Referral Code
After enrolling in any paid course, students automatically receive a unique referral code (format: STU-XXXXXXXX) visible in their dashboard.
Share with Friends
Student shares the code via social media, messaging apps, email, or word of mouth. A shareable referral link is also provided.
Friend Signs Up
Friend creates an account on Online IBS using the referral code. The referral is tracked but credit is not yet awarded.
Friend Enrolls in Paid Course
Friend browses courses and completes payment for any paid program. This triggers the referral reward process.
Credit Awarded to Referrer
The referring student receives ‚Ç¨20-35 in credits (based on their referral tier). Credits are valid for 60 days from award date.
Use Credits at Checkout
Student can apply credits during checkout. Full balance usable for courses; 50% limit for shop products.

  • Courses: 100% ‚Äì Full credit balance can cover course fees
